## 👋 About me

I'm **StraycoderX**, a programming enthusiast focused on **Python**, **C++** and **Node.js**.
I build **CRUD APIs**, backend services and **web-security tooling** — I enjoy projects that mix clean
software design with a security mindset. Open to collaboration: feel free to open an issue or reach out.

- 🔭 Currently working on: API integration, databases and security automation
- 🌱 Learning: secure backend architecture, CI/CD and static analysis
- 🛡️ Flagship project: **[Vulnerabilidades](https://github.com/StraycoderX/Vulnerabilidades)** — a dependency-free web vulnerability scanner (CI + CodeQL + Docker + reusable Action)
- 💬 Ask me about: Python, REST APIs, web security
- 🌍 Languages: Español / English
